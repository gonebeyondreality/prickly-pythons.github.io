Welcome to the SESE Python Users Group

This group was created as a discussion and sharing platform for Python users and learners at the School of Earth and Space Exploration (SESE), Arizona State University. We use GitHub Organization platform due to its integration of user management, version control, and discussion platform via issue tracking.
